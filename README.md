## run in py venv
```
# 建立一個名為 venv 的虛擬環境
python3 -m venv venv

# 啟動虛擬環境（bash/zsh）
source venv/bin/activate

# 在虛擬環境中安裝 selenium
python3 -m pip install selenium

# 然後執行您的腳本
python3 mooc_auto.py
```
